Meghan Markle and Employees API

This API lists Meghan Markle's movies, and also some employees from a place:
Route Method Description Example 
/markle
 GET Gets a list of Meghan Markle's movie titles and ids List Movies 
/markle/:id
 GET Returns the detail about the movie matching the id in the parameter Get Movie 1
Route Method Description Example 
/employees
 GET Gets a list of employee names and ids List Employees 
/employees/:id
 GET Returns the detail about the employee matching the id in the parameter Get Employee 1
